# classhub_client

## Overview

**classhub_client** is a comprehensive front-end application for managing classrooms and users in an educational setting. It features role-based access controls, ensuring that Principals, Teachers, and Students have appropriate access to functionalities like user management, classroom assignments, and timetable creation.

## Features

- **Principal Dashboard**: Manage teachers, students, and classrooms.
- **Teacher Dashboard**: Manage students and create timetables.
- **Student Dashboard**: View classmates and timetables.
- **Authentication**: Secure login and signup processes with role-based routing.
- **Responsive Design**: Built with Tailwind CSS and DaisyUI for a seamless user experience across devices.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- A running instance of the backend server (`classhub_server`)
